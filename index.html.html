Web Application Development Proposal

Ad-Reward Platform - MVP

1 Project Overview

This document outlines the scope, features, timeline, and pricing for the development of a mobile-first web application that enables users to earn rewards by viewing advertisements and convert those rewards into real money, which can be withdrawn to their bank accounts.

The application is designed as an MVP (Minimum Viable Product) to validate the business model and can be scaled with advanced features in later phases.

2 Application Concept

  • Users register on the platform and purchase a subscription plan
  • Based on the selected plan, users can watch advertisements from different categories
  • Each completed advertisement credits reward points to the user's wallet
  • Accumulated points can be converted into monetary value
  • Users can request withdrawals to their linked bank accounts
  • An admin panel manages users, ads, subscriptions, and payouts
